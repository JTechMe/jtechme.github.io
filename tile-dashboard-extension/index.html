Chromium Browsers

For Chrome, Edge, Brave, Opera, and other Chromium-based browsers:

  • Download the .crx file above
  • Open chrome://extensions/ in your browser
  • Enable "Developer mode" (top-right toggle)
  • Drag and drop the .crx file onto the page
  • Click "Add extension" when prompted
  • Open a new tab to enjoy your dashboard!
🦊

Firefox Browser

For Mozilla Firefox users:

  • Download the .xpi file above
  • Open Firefox and navigate to about:addons
  • Click the gear icon and select "Install Add-on From File"
  • Select the downloaded .xpi file
  • Click "Add" when prompted
  • Open a new tab to see your new dashboard!
